Shaping Regions: Hint 1

An array of all 'points' is too big; 16MB maximum.

Keep track of the rectangles' coordinates; split the rectangle when an overlap occurs, e.g.: 

+--------+      +-+--+--+
|        |      | |2 |  |
|        |      + +--+  |
|  +-+   |  --> | |  |  |
|  +-+   |      |1|  |3 |
|        |      | +--+  |
|        |      | | 4|  |
+--------+      +-+--+--+
