ACM-ICPC Indonesia National Contest 2012

Problem A

The Greatest Man in The Universe

Time Limit: 2s

Given two integers A and B, and a sequence of positive integers S0..N-1 which consists of N elements, determine the length of the longest subsequence which should be chosen by the following rule:

  1. Choose i from 0 to N-1.
  2. If Si is between A and B (inclusive), then:
    • Take Si to be part of the chosen subsequence.
    • Update i with i + Si.
    Otherwise, stop.
  3. Repeat step 2 until i ≥ N or Si is not in between A and B.

The length of a subsequence is defined as the number of elements in it.

*) In case you don't know what "inclusive" means, we say an integer P is between A and B inclusively is A ≤ P ≤ B is satisfied.


For example, let S contains 9 elements: {1, 4, 2, 3, 2, 7, 1, 2, 3}, A = 1 and B = 4. There are 9 possibilities:

  • start with i = 0 : 1, 4, 2, 3, 2, 7, 1, 2, 3; resulting a subsequence {1, 4} with size of 2.
  • start with i = 1 : 1, 4, 2, 3, 2, 7, 1, 2, 3; resulting a subsequence {4} with size of 1. The second element is 7 which is not between A and B, so we stop at 4.
  • start with i = 2 : 1, 4, 2, 3, 2, 7, 1, 2, 3; resulting a subsequence {2, 2, 1, 2} with size of 4.
  • start with i = 3 : 1, 4, 2, 3, 2, 7, 1, 2, 3; resulting a subsequence {3, 1, 2} with size of 3.
  • start with i = 4 : 1, 4, 2, 3, 2, 7, 1, 2, 3; resulting a subsequence {2, 1, 2} with size of 3.
  • start with i = 5 : 1, 4, 2, 3, 2, 7, 1, 2, 3; resulting a subsequence { } with size of 0. The integer 7 cannot be used because it is not between A and B, so we stop there and the result is a subsequence with size of 0.
  • start with i = 6 : 1, 4, 2, 3, 2, 7, 1, 2, 3; resulting a subsequence {1, 2} with size of 2.
  • start with i = 7 : 1, 4, 2, 3, 2, 7, 1, 2, 3; resulting a subsequence {2} with size of 1.
  • start with i = 8 : 1, 4, 2, 3, 2, 7, 1, 2, 3; resulting a subsequence {3} with size of 1.

Out of those 9 possibilities, the longest one is {2, 2, 1, 2} which has size of 4.

Given S, A and B, your task is to determine the length of the longest subsequence following the above rule. The title? oh sorry, it has nothing to do with this problem.


Input

The first line of input contains an integer T (1 ≤ T ≤ 100) the number of cases. Each case begins with three integers N (1 ≤ N ≤ 100), A and B (1 ≤ A ≤ B ≤ 100) denoting the size of sequence S, A and B which described at the problem statement respectively. The next line contains N integers ranged between 1 and 100 (inclusive) denoting the sequence S.

Output

For each test case, output in a line the "Case #X: Y" where X is the case number starts from 1, and Y is the length of the longest subsequence that can be chosen by following the rule.
