ACM-ICPC Indonesia National Contest 2012

Problem F

N Choose K

Time Limit: 2s

Given N integers a1, a2, a3 ... aN-1, aN; you have to choose K integers from those integers and relabel them into b1, b2, b3 ... bK-1, bK. Those chosen K integers should have ALL of the following properties:

  • b1 + b2 + b3 + ... + bK-1 > bK
  • b1 + b2 + b3 + ... + bK-2 + bK > bK-1
  • b1 + b2 + b3 + ... + bK-3 + bK-1 + bK > bK-2
  • ...
  • ...
  • b1 + b3 + ... + bK-1 + bK > b2
  • b2 + b3 + ... + bK-1 + bK > b1

For example, let N = 5, K = 3 and the integers are: {3, 1, 7, 5, 3}. We can choose {3, 7, 5} as they satisfy the required property:

  • 3 + 7 > 5
  • 3 + 5 > 7
  • 5 + 7 > 3

Your task is easy, you only have to determine whether it is possible or not to find such K integers from the given N integers. Output "YES" (without quote) if it is possible, otherwise "NO" (without quote).


Input

The first line of input contains an integer T (1 ≤ T ≤ 100) the number of cases. Each case begins with two integers N (3 ≤ N ≤ 100) and K (3 ≤ K ≤ N) as defined above. The next line contains N integers separated by a single space.
Each integer is non-negative and not more than 2,000,000,000.

Output

For each test case, output in a line the "Case #X: Y" where X is the case number starts from 1, and Y is "YES" or "NO" (without quote) which indicate whether you can choose K integers with property described above or not.
