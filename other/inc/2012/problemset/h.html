ACM-ICPC Indonesia National Contest 2012

Problem H

Good Team Work

Time Limit: 2s

ICPC 2012 is drawing near and UCA (University of Cool and Awesomness) is going to send some of their teams to compete in this prestigious programming contest event. As you might alread know, each team consists of 3 students.

There are 3N students and Mr. Rhino -the team's coach- wants to form exactly N teams such that all students get their own team. From the past year experience, Rhino realized that team work is an important factor to consider when forming a team, that's why Rhino assesses the compability for each pair of student. The team-compability value is defined as the total compability between each pair of student in the team. This year, instead of trying to win ICPC (by forming one best team, and let the rest do their own), Rhino wants to create teams with good team work. So, all of his N teams, he wants the teams are arranged such that the lowest team-compability value is as high as possible.

Given a matrix of 3N x 3N, help Rhino to find the highest possible value for the lowest team-compability of his teams.


Input

The first line of input contains an integer T (1 ≤ T ≤ 100) the number of cases. Each case begins with an integer N (1 ≤ N ≤ 5). The next 3N lines each contains 3N integers aij (0 ≤ aij ≤ 1,000) denoting the compability value of ith and jth student. aij will be equal to aji.

Output

For each test case, output in a line the "Case #X: Y" where X is the case number starts from 1, and Y is the highest possible value for the lowest team-compability of Rhino's teams in each case.
